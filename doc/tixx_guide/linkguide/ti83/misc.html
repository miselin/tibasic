Deleting a variable

In fact, Ti-Connect use an ersatz for this: it sends a key sequence by remote control which does the following:
- returns to HOME
- go to CATALOG
- select DelVar
- type 'varname'
- type ENTER

Key sequence is (for example, variable is 'B') : 0040 (QUIT), 0009 (CLEAR), 0009 (CLEAR), 003E (CATALOG), 009D (D), 0004 (DOWN), 0004 (DOWN), 0004 (DOWN), 0005 (ENTER), 0009B (B), 0005 (ENTER).
